About This Project

It is a FYP project of Information Engineering from City University of Hong Kong. NewsPoint, which is my developed web application, is the combination of text mining concepts, NLP, Frontend and Backend Databse server design.

News Summarizer

01

Using TF*IDF and Training Model to generate an extractive and abstractive summary, extract the key words and concepts from text.

Text Analyzer

02

Using statistic Approach to analysis the structure and readability from the newspaper with a graphical dashboard virtualization.

Web Scraping

03

With the API from Google news database and web mining, it can help you to extract different news time to time.

Machine Learning

04

With supervisored learning, the system can distinguish the type of sentence and the importance of words.

Input Method

05

Different Input Design, including Text Input, File Input, and the web mining from the input URL to extract the news article.

Topic Modelling

06

The system can classify different topic and emotion from the newspaper. It can also recommend some interesting news to you.
